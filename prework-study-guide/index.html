HTML

  • HTML - creates content. text, images, parts of a page's layout
  • Element types used to add content to pages: Text elements, Image elements, Header elements, List elements,
  • Element types used to divide and describe page: Meta-related elements, Content division elements, Semantic elements,
  • head element contains information, or metadata, that helps the browser to render the page correctly. Elements included within head are not visible when a user looks at a webpage.
  • body element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element
  • head element is a container for metadata
  • Metadata - data about our webpage that the browser uses to effectively serve our webpage to users
  • head: contains information about the webpage
    !DOCTYPE html - declaration that tells browser what document types
    html - start of html document
    lang="en" - tells browser language for website, en for English
    title - defines the title for a webpage
    UTF-8 - allows us to use most characters and symbols on the page
    /head - closing tag
    link - tag defines the relationship between the current document and an external resource
  • opening tag and closing tag - We insert the content between an opening tag and a closing tag so that the browser understands how to treat the content
    opening tag - head
    closing tag - /head
  • body: all content you want to display to the user must be inside the body element
    h1, /h1 - the content between these tags is what is rendered on the page
    h1, h2, h3, h4, h5, h6 - represent the level of heading a given text block represents
    p - represents a paragraph or block of text
    ul, ol, li - represent unordered lists, ordered lists, and list items. These HTML elements represent bulleted or numbered lists
    img - contains information about images that are displayed on the webpage
    ----src - defines the location of the image
    ----alt - contains a text string that describes the appearance and functionality of the image
    br - creates a line of empty space, or a line break, between two blocks of content
    a (anchor) - creates links to the same webpage or other webpages
    ----href - points to the URL for the link
    header - appears at the top of the page and usually contains a navigation menu and/or introductory content
    footer - appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation
    section - semantic elements that define a section in a document that contains thematic content
  • Semantic HTML elements - the name of the element describes its content and the role it plays on a webpage
  • Child elements - elements nested inside other elements
  • Parent element - element that holds all the child elements

CSS

  • CSS - Adds styling to the content. Fonts, colors, spacing, etc.
  • margin - Indicates how much space we want around the outside of an element.
  • padding - Indicates how much space we want around the content inside an element.
  • width - Assigns width to an element.
  • height - Assigns height to an element.
  • background-color - Assigns background color to an element.
  • color - Assigns color to an element.
  • display - Specifies how an element is shown on a web page.
  • font-size - Assigns a font size to an element.
  • border - Assigns a border with certain width.
  • box-shadow - CSS property adds shadow effects around an element's frame.
  • text-align - Positioning property. One way to center a title or heading on a webpage
  • Inline CSS - When a developer wants to style an element directly in the HTML file.
    Internal CSS style sheet - Contains CSS rules for the webpage in the head section of the HTML file.
    External CSS style sheet - Allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • selector{property: value; property: value;}
  • 2 or more Selectors - Combine two selectors into a single CSS rule with a comma, allowing the declarations to affect both selectors.
  • Declarations - Contain two important components: the CSS property we want to apply and the value of the property.
  • * - Apply a rule to all the elements visible on the page, wild card symbol.
  • Class attribute - allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • CSS box model - Each box consists of margins outside the element, and the element's borders, padding, and content.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: command to commit all files in the working branch
  • git commit -m "": We use the -m flag to associate a message with our commit. Commit messages should be short descriptive messages of the changes implemented
  • git pull origin main: command to pull in the base branch
  • git push origin: push up all of our changes

JavaScript

  • JavaScript - Add logic to webpage, which allows users to interact with it.
  • variables - Containers that store values. You start by declaring a variable with the let keyword, followed by the name you give to the variable.
    String - This is a sequence of text known as a string. To signify that the value is a string, enclose it in single or double quote marks.
    Number - This is a number. Numbers don't have quotes around them.
    Boolean - This is a True/False value. The words true and false are special keywords that don't need quote marks.
    Array - This is a structure that allows you to store multiple values in a single reference.
    Object - This can be anything. Everything in JavaScript is an object and can be stored in a variable. Keep this in mind as you learn.
    Null - Inhabited by exactly one value: null.
    Undefined - Inhabited by exactly one value: undefined.
    Bigint - A numeric primitive in JavaScript that can represent integers with arbitrary magnitude. With BigInts, you can safely store and operate on large integers even beyond the safe integer limit for Numbers.
    Symbol - A unique and immutable primitive value and may be used as the key of an Object property.
  • const - Allows us to create variables that can't be reassigned a value.
    let - Allows us to reassign the value of the variable over time.
  • console.log() - Used to output a message to the web console by adding an argument.
  • ; - A semicolon at the end of a line indicates where a statement ends.
  • /*
    this is a comment
    for multiple lines
    */
    // this is a comment for one line
  • Operators:
    Addition: +
    Subtraction: -
    Multiplication: *
    Division: /
    Assignment: =
    Strict equality: ===, test to see if two values are equal. Returns a true/false (Boolean) result.
    Not: !, this returns the logically opposite value of what it precedes.
    Does-not-equal: !==
  • Control flow - The order in which a computer executes code in a script. In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • Conditional statement - Control behavior in JavaScript and determine whether or not pieces of code can run.
    "if" statement: A way of introducing decision-making into our code. Just like its name sounds, when using the "if" statement, code will only execute if the statement meets a condition, or is true.
    "for" loop: Uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
    const shapes = ["triangle", "square", "pentagon", "circle"];
    for(let x = 0; x < shapes.length; x++) {
    console.log(shapes[x]);
    }
  • Array - A single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way. Use: []
    Index number: 0, 1, 2, 3, 4...
    .length - Can find the length of an array.
  • Function - A set of instructions that tells the computer how to perform a certain task. To execute a function, we must call it in our code.
    function helloWorld() {
    console.log("Hello, world, I am a function!")
    }
  • Click event - fired by the browser when you click on something with your mouse.
