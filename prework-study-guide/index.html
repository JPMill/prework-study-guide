HTML

  • HTML - creates content. text, images, parts of a page's layout
  • Element types used to add content to pages: Text elements, Image elements, Header elements, List elements,
  • Element types used to divide and describe page: Meta-related elements, Content division elements, Semantic elements,
  • head element contains information, or metadata, that helps the browser to render the page correctly. Elements included within head are not visible when a user looks at a webpage.
  • body element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element
  • head element is a container for metadata
  • Metadata - data about our webpage that the browser uses to effectively serve our webpage to users
  • head: contains information about the webpage
    !DOCTYPE html - declaration that tells browser what document types
    html - start of html document
    lang="en" - tells browser language for website, en for English
    title - defines the title for a webpage
    UTF-8 - allows us to use most characters and symbols on the page
    /head - closing tag
  • opening tag and closing tag - We insert the content between an opening tag and a closing tag so that the browser understands how to treat the content
    opening tag - head
    closing tag - /head
  • body: all content you want to display to the user must be inside the body element
    h1, /h1 - the content between these tags is what is rendered on the page
    h1, h2, h3, h4, h5, h6 - represent the level of heading a given text block represents
    p - represents a paragraph or block of text
    ul, ol, li - represent unordered lists, ordered lists, and list items. These HTML elements represent bulleted or numbered lists
    img - contains information about images that are displayed on the webpage
    ----src - defines the location of the image
    ----alt - contains a text string that describes the appearance and functionality of the image
    br - creates a line of empty space, or a line break, between two blocks of content
    a (anchor) - creates links to the same webpage or other webpages
    ----href - points to the URL for the link
    header - appears at the top of the page and usually contains a navigation menu and/or introductory content
    footer - appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation
    section - semantic elements that define a section in a document that contains thematic content
  • Semantic HTML elements - the name of the element describes its content and the role it plays on a webpage
  • Child elements - elements nested inside other elements
  • Parent element - element that holds all the child elements
